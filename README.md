## Démarrage

```bash
git clone git@github.com:IIM-Creative-Technology/vuejs-sacha0659.git
```

```bash
npm install 
```

```bash
npm run serve 
```

## Différentes routes

- "/admin" => Permet la gestion des posts (création) 
- "/blog" => Affichage des différents posts
